Prerequisitos para utilizar el sistema MedicalVM:

  • Será necesario tener instalado un navegador Netscape 4.7 o Internet Explorer versión 5.0 o superior.
  • Además, deberá tener habilitada la opción Javascript, para ello:
    • En Netscape vaya a Edit/Preferences/Advanced y asegúrese que tiene la opción Enable Javascript activada
    • En Explorer vaya a Herramientas/Opciones de Internet/Seguridad y asegurese que el nivel de seguridad en la zona Internet es: mediano o inferior.
